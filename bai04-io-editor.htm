2. nano

L phn mm son tho vn bn trong mi trng text ph bin trong nhiu distro linux khc nhau. nano c ci t rng ri v s nh gn, d s dng. Tt c cc menu lnh, hi p vi ngi dng c th hin ti hai dng cui cng ca ca s lnh. Ngi dng ch cn hiu ^X tng ng vi nhn Ctrl v X l c th thao tc d dng vi nano.

3. vi

Phn mm son tho vn bn ni ting nht trong mi trng ix. Rt mnh v cng rt kh iu khin tuy nhin 100% nhng ngi dng Linux chuyn nghip u thnh tho vi. Trong cc distro CentOS 6.x, vi c hai phin bn vim-minimal c cc tnh nng c bn v vim-enhanced vi cc tnh nng nng cao. Theo mc nh, s dng bn minimal ngi dng cn chy lnh vi v s dng bn enhanced cn chy lnh vim. Tuy nhin trong phng thc hnh, ngi dng root ti cc my o c thit lp sn mt  alias vi=vim   lun s dng vim-enhanced bng lnh vi.

a. Cc ch hot ng

vi c nhiu ch hot ng khc nhau, tuy nhin ti liu ny ch gii thiu hai ch hot ng c bn nht:

Ch lnh

L ch mc nh khi bt u khi ng vi. Trong ch ny, ngi s dng ra cc lnh iu khin v vi thc hin theo. Ngi dng c th thay i ni dung vn bn cn son tho nhng khng th d dng nhp ni dung mi nh trong cc trnh son tho vn bn khc.

Ch son tho

Biu hin bng on k t  --INSERT--  ti gc di bn tri mn hnh. Trong ch ny, bn phm c s dng nhp vn bn nh cc trnh son tho vn bn thng dng  khc.

chuyn sang ch ny t ch lnh g phm  i  .

quay v ch lnh g phm  Esc  trn bn phm.

b. Di chuyn, tm kim trong vi

4. mcedit

L mt phn ca b cng c ni ting mc (Midnight Commander). mcedit cho php lm vic kh tt vi nhiu loi file. Vi h thng menu giao din mnh, mcedit rt thun tin trong son tho v c bit mnh khi lm vic vi khi.

II. Chuyn hng kt xut

Mc nh mi tin trnh trong Linux khi c np vo b nh u c HH cp pht cho ba knh nhp-xut d liu chun c th lm vic m khng cn quan tm n nhp-xut d liu. Knh nhp chun l bn phm v knh xut chun l mn hnh. Vic chuyn knh mc nh c thc hin bi cc k hiu chuyn knh nu di y. Nu c nhiu yu cu chuyn knh trong mt ln gi, HH s chuyn knh theo th t u tin t phi sang tri ca dng lnh.

1. stdin

0<         hoc vit tt l          <

c s dng thay i stdin mc nh ca tin trnh l bn phm thnh file c ch nh.

2. stdout

1>         hoc vit tt l          >

c s dng thay i stdout mc nh ca tin trnh l mn hnh thnh file c ch nh. File ny nu c s b xa ht ni dung trc khi nhn output ca tin trnh.

1>>       hoc vit tt l          >>

c s dng thay i stdout mc nh ca tin trnh l mn hnh thnh file c ch nh. File ny nu c ni dung s ghi output ca tin trnh vo sau ni dung c c.

|

c s dng thay i stdout mc nh ca tin trnh l mn hnh thnh stdin ca tin trnh c ch nh sau k hiu chuyn hng ny. C ch chuyn hng ny cn gi l c ch ni dng, l linh hn ca cc shell script s dng trong qun l h thng.

3. stderr

2>

c s dng thay i stderr mc nh ca tin trnh l mn hnh thnh file c ch nh. File ny nu c s b xa ht ni dung trc khi nhn warning hoc error ca tin trnh.

2>>

c s dng thay i stderr mc nh ca tin trnh l mn hnh thnh file c ch nh. File ny nu c ni dung s ghi warning hoc error ca tin trnh vo sau ni dung c c.

2>&1

c s dng thay i stderr mc nh ca tin trnh l mn hnh thnh stdout.

Bi 4: Thc hnh chuyn hng kt xut (1)

Chy lnh sau  cat > /tmp/new.txt  g mt s on k t c xung dng ri nhn Ctrl+D kt thc nhp. Gii thch cc thao tc trn

To file sum.sh c ni dung nh sau:

 

 echo n Nhap A: ; read A

 echo n Nhap B: ; read B

 echo A + B = $(($A + $B))

 

Sau thc hin cc lnh sau chy file mi to

 

 chmod a+x sum.sh

 ./sum.sh

 

Nhp A, B bng bn phm tnh A + B. Sau to file input.txt c ni dung nh sau (lu c 3 dng, dng th 3 trng)

 

 16

 27

 

Sau chy lnh sau kim tra chuyn hng u vo (stdin)

 

 ./sum.sh < input.txt

 

Bi 5: Thc hnh chuyn hng kt xut (2)

 cat nofile 2>&1 > output.txt
