Mount

mount <tn ph.vng> <th mc> [-t kiu] [-o t.chn kiu]

Thit b phi c gn vo 1 th mc trng bt k c sn trn cy th mc trc khi bn c th truy cp ti n. Th mc trng m gn vi thit b k trn c gi l mount point.

Sau khi mount thit b, bn c th truy cp ti d liu trong thit b bng cch truy cp vo mount point. Bn cn thc hin thao thc unmount hy gn kt thit b vi h thng khi khng cn cn truy cp ti thit b na.

Ty chn kiu ca lnh mount ging ht ty chn kiu c s dng trong file /etc/fstab.

nh x t ng qua /etc/fstab (mount a)

File cu hnh /etc/fstab cha thng tin v cc thit b (phn vng cng, CD/DVD, USB, ISO image) trn my tnh bao gm:

+ ng dn ti file i din cho thit b.

+ Mount point: cho bit thit b c mount vo th mc no.

+ Cc ty chn (option): ch ra thit b c mount nh th no?

v.v..

fstab l file dng vn bn (plain text), v th bn c th m v chnh sa n s dng bt k cng c Text Editor no vi iu kin bn phi c c quyn root lu li nhng thay i. Nu bn ang ng nhp vi ti khon ngi dng thng thng th c th s dng 2 lnh su hoc sudo tm thi chuyn sang c quyn root.

Xem chi tit v /etc/fstab bng man 5 fstab

Mi dng trong file fstab cha thng tin v mt thit b. Cc ct mi hng c phn cch bi khong trng. Th t cc dng l khng quan trng. Sau y phn gii thch k hn ngha ca tng ct:

+ Ct 1: cho bit loi thit b (phn vng, CD/DVD, USB, ISO image). ng thi cng cho bit ng dn ti file i din cho thit b (device file) . Trong Linux, mi ti nguyn phn cng ln phn mm u c xem l file, cc device file thng nm th mc /dev

+ Ct 2: ng dn ca mount point, l mt th mc trng c to sn trong cy th mc. Khi g lnh mount, nu bn khng ch nh r mount point th y l mount point mc nh cho thit b ct 1. Th mc cha mount point thng l /mnt hay /media , mc d bn c th mount thit b vo bt c th mc trng no.

+ Ct 3: l kiu filesystem ca thit b. Linux h tr nhiu kiu filesystem, di y l 1 s filesystem ph bin:

         Ext2, Ext3 v Ext4: im khc bit ln nht gia Ext2 v Ext3 (Ext4) l Ext3(4) h tr tnh nng journaling, tc l khi bn tt my khng ng cch (do cp in t ngt, h thng b treo nn phi nhn nt khi ng li) th kh nng mt mt d liu mc thp, ng thi h iu hnh s khng phi tn thi gian kim tra, tm li trn filesystem trong ln khi ng k tip.

Xem thm ti http://en.wikipedia.org/wiki/Ext4

         swap: phn vng lm khng gian b nh o, dng b sung thm b nh cho h thng khi h iu hnh pht hin vic thiu ht b nh RAM.

         Vfat (FAT16, FAT32) v NTFS: y l cc filesystem c Windows h tr.

         nfs: dnh cho cc ti nguyn xa, c chia s qua mng s dng NFS

         auto: y khng phi l 1 filesytem. N c ngha l h thng s t ng nhn din loi filesystem ca thit b khi thit b c mount.

+ Ct 4: l cc ty chn khi mount.

Nu c nhiu ty chn th chng c phn cch bi du phy. Di y l 1 s ty chn ng ch :

         auto: t ng mount thit b khi my tnh khi ng.

         noauto: khng t ng mount, nu mun s dng thit b th sau khi khi ng vo h thng bn cn chy lnh mount.

         user: cho php ngi dng thng thng c quyn mount.

         nouser: ch c ngi dng root mi c quyn mount.

         exec: cho php chy cc file nh phn (binary) trn thit b.

         noexec: khng cho php chy cc file binary trn thit b.

         ro (read-only): ch cho php quyn c trn thit b.

         rw (read-write): cho php quyn c/ghi trn thit b.

         sync: thao tc nhp xut (I/O) trn filesystem c ng b ha.

         async: thao tc nhp xut (I/O) trn filesystem din ra khng ng b.

         defaults: tng ng vi tp cc ty chn rw, suid, dev, exec, auto, nouser, async

+ Ct 5 l ty chn cho chng trnh dump, cng c sao lu filesystem. in 0: b qua vic sao lu, 1: thc hin sao lu.

+ Ct 6 l ty chn cho chng trnh fsck, cng c d li trn filesystem. in 0: b qua vic kim tra, 1: thc hin kim tra cho root, 2: thc hin kim tra cho cc filesystem khc.

File /etc/fstab cung cp cc ch dn cho h iu hnh trong vic nhn din, qun l vic mount cc thit b. ng thi vic cu hnh li file fstab cng gip ch cho bn trong vic gim bt thi gian mount thit b bng lnh cng nh l kim sot vic truy cp ti thit b ca ngi dng.

Mt s ch vi mount v /etc/fstab

Chng trnh mount s c /etc/fstab v quyt nh ti nguyn (hoc im kt ni) no s c s dng v cc tham s ca vic kt ni cng c th c xc nh ti bc ny. Cc tham s ca mount v trong file fstab l ging nhau.

Nu s dng vi tham s -a, mount s t ng nh x tt c cc khai bo trong /etc/fstab m cha c mount v khng c ty chn noauto.

LABEL and UUID

Mt s thit b c truy cp thng qua cc nhn hoc UUID (Universally Unique Identifier).

Nhn c gn cho thit b bi lnh tune2fs:

             tune2fs L /usr/local /dev/hdb12

Nhn c gn cho thit b bi lnh e2label:

             e2label device [new-label]

UUID c gn cho thit b bi lnh tune2fs:

tune2fs U random <tn ph.vng>

Xem UUID ca phn vng

tune2fs l <tn ph.vng> | grep UUID

Remount

Khi mun thay i cc option ca mt filesystem c mount, chng ta khng cn thit phi unmount ri mount li vi cc option mi m c th dng lnh mount vi tham s remount m khng cn phi thay i tn phn vng hay mount point.

Thng th chng ta remount chuyn mt filesystem t ch ch c sang c v ghi nh sau:

mount -o remount,rw /dev/sdb2

II. Quota

1. Khi nim Hn ngch

Quota l cng c cho php qun tr h thng thit lp hn ngch lu tr trn a. Cng c ny khng yu cu khi ng li h thng.

C hai loi qun l hn ngch c bn: theo dung lng a cng s dng (block quota) v theo s file v th mc c to ra (inode quota).

-          Block quota s tnh tng dung lng ca tt c cc file v th mc ca ngi dng to ra trong filesystem theo s cc block. Kch thc mc nh ca cc block khc nhau trong tng h thng file..

-          Inode quota s tnh tng s lng file v th mc ca ngi dng c to ra trong h thng file (mi file v th mc thng chim 1 inode).

Hn ngch a thng c p dng cho ngi dng hoc nhm ngi dng. Vic t ra hn ngch s do ngi qun tr h thng thc hin cho tng ngi dng hoc nhm ngi dng.

C hai loi mc hn ngch (cho c block v inode): mm (soft) v cng (hard). Thng thng th soft quota < hard quota. Khi soft quota b vi phm, h thng s gi mt cnh bo n ngi dng. Khi hard quota b vi phm, h thng s ngn chn hnh ng ghi d liu ca ngi dng vo h thng file v nhn mt thng bo v vic ghi khng thnh cng.

2. Cc bc ci t quota cho ngi dng

c th p dng tnh nng hn ngch trn h thng file, cn phi lm cc bc sau.
