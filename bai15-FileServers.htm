Bi 15. File Servers

1.           FTP Server

1.1. Gii thiu

File Transfer Protocol (FTP) l giao thc cho download/upload tp. FTP lm vic theo m hnh client/server. FTP server cn c gi l FTP daemon. N lin tc lng nghe cc yu cu FTP t cc clients. Khi nhn c mt yu cu, FTP server thc hin th tc ng nhp v thit lp kt ni vi mt phin lm vic. Trong sut phin lm vic, FTP server nhn v thc yhi bt k lnh no c gi n bi client.

C th truy cp FTP server theo hai ch : Anonymous (V danh) v    Authenticated (C xc thc). ch v danh, clients c th truy cp FTP server bng ti khon mc nh c tn l "ftp" v dng a ch email lm mt khu. ch xc thc, ngi dng phi c ti khon v mt khu h thng truy cp FTP server.

FTP dng 2 knh TCP: cng 20 cho d liu v cng 21 cho lnh. Tin trnh giao tip cng 21 gi v nhn lnh, c gi l  protocol interpreter, hay PI. Tin trnh giao tip cng 20 thc hin truyn/nhn d liu c gi l  data transfer process, hay DTP.

FTP khc vi tt c ng dng TCP/IP l phi dng 2 knh, cho php gi lp truyn ti lnh v d liu ca FTP. N cng khc cc giao thc khc mt chc nng quan trng: FTP truyn ti files foreground, so vi giao thc khc background. Hay ni cch khc, FTP khng dng spoolers hay queues, do vy vic truyn ti thc hin bng thi gian thc.

1.2. Ci t v cu hnh

FTP deamon cho Ubuntu c tn l vsftpd. Chy lnh sau ci t vsftpd:

sudo apt install vsftpd

Tp cu hnh vsftpd l /etc/vsftpd.conf.

Ch v danh

  s dng ch v danh, chnh sa tp /etc/vsftpd.conf bng thay i sau:

anonymous_enable=Yes

ch v danh, ngi dng ftp c s dng. Trong qu trnh ci t, ngi dng ftp c to, cng vi th mc home cho ngi dng ny l /srv/ftp. Nu mun thay i th mc home cho ftp, v d thnh /srv/files/ftp, hy chy cc  lnh sau:

sudo mkdir /srv/files/ftp

sudo usermod -d /srv/files/ftp ftp

Nu mun cho ngi dng v danh (ftp) c upload tp, t tham s sau:

anon_upload_enable=YES

Lu , cho upload tp ch v danh c ri ro an ninh rt cao.

Cn khi ng li vsftpd n nhn cu hnh mi.

sudo restart vsftpd

Ch c xc thc

  s dng ch c xc thc, chnh sa tp /etc/vsftpd.conf bng thay i sau:

anonymous_enable=No

ch xc thc, ngi dng s dng ti khon h thng ng nhp FTP Server.

Nu cho php ngi dng upload tp, t thng s sau:

write_enable=YES

Cn khi ng li vsftpd n nhn cu hnh mi.

sudo restart vsftpd

1.3. Thit lp an ninh

gii hn ngi dng ch c truy cp trong th mc home ca h, s dng gi tr cu hnh sau:

chroot_local_user=YES

Cng c th danh sch ngi dng ch c quyn trong th mc home ca h bng cc tham s:

chroot_list_enable=YES

chroot_list_file=/etc/vsftpd.chroot_list

To tp /etc/vsftpd.chroot_list cha danh sch ngi dng, mi ngi dng trn mt dng.

khng cho mt ngi dng c s dng FTP, hy a tn ngi dng  vo

tp /etc/ftpusers. Danh sch mc inh bao gm root, daemon, nobody.

C th s dng FTP trn Secure Socket Layer (SSL), tc FTPS. Cu hnh bng cch thm dng sau vo cui tp /etc/vsftpd.conf.

ssl_enable=Yes

ng thi lu chng th v kha lin quan:

rsa_cert_file=/etc/ssl/certs/ssl-cert-snakeoil.pem

rsa_private_key_file=/etc/ssl/private/ssl-cert-snakeoil.key

Theo mc nh, FTP server s dng chng th v kha c cung cp bi gi ssl-cert package. Trong mi trng sn xut, hy thay th bng chng th v kha c to cho host.

Lu cn khi ng li vsftpd n nhn cu hnh mi.

2.           NFS

2.1. Gii thiu

NFS (Network File System) l h thng cung cp dch v chia s tp ph bin hin nay trong h thng mng Linux v Unix. NFS s dng m hnh Client/Server. NFS Server c cc a cng vt l cha cc file system c chia s v mt s dch v chy ngm trn h thng (daemon) phc v cho vic chia s vi NFS client. Ngoi ra, cc dch v chy trn NFS Server cng cung cp chc nng bo mt tp v qun l lu lng s dng file system quota.

NFS clients mun s dng cc file system c chia s trn NFS Server ch n gin dng giao thc NFS mount cc file system ln h thng ca mnh.

2.2. Ci t v cu hnh NFS Server

Chy lnh sau ci t NFS

sudo apt install nfs-kernel-server

C th cu hnh cc th mc c chia s bng vic thm th mc vo tp /etc/exports. V d

/ubuntu  10.0.0.0/24 (ro,sync,no_root_squash)

/home    *(rw,sync,no_root_squash)

Trong :

o   /ubuntu v /home: Th mc cn chia s

o   10.0.0.0/24 hay *: Di a ch IP m NFS server cho php truy cp

o   ro: Ch cho c v khng cho ghi vo th mc

o   rw: Cho php ghi vo th mc chia s

o   sync: synchronize

o   no_root_squash: cho quyn root

o   no_all_squash: s dng quyn ca ngi dng

Khi ng li NFS server:

sudo systemctl start nfs-kernel-server.service

2.3. Ci t v cu hnh NFS client

Trn my mun s dng th mc c chia s bi NFS server, ci t nfs-common bng lnh:

sudo apt install nfs-common

S dng lnh mount nh x th mc c chia s bi NFS server vo th mc trn client:

sudo mount nfsserver.hostname:/ubuntu /local/ubuntu

 C th thc hin mout t ng bng cch thm dng cu hnh vo tp /etc/fstab

nfsserrver.hostname:/ubuntu /local/ubuntu nfs rsize=8192,wsize=8192,timeo=14,intr
